=> 선언한 추상 메서드는 구현 클래스(하위 클래스)를 만들어서 구현한다. =>
 이떄 사용하는 것은 imple x => extends를 사용해야 한다.

구현체로 만들고 객체 생성 가능
추상 메서드 자체의 객체 생성은 불가능

구현 클라스는 추상 클라스의 메서드를 필수적으로 구현해야 한다.


1. 메서드가 없이 클래스만 존재하는 추상 메서드 만들기는 가능하다.

2. 추상 클래스를 extends하는 또 다른 추상 클래스2를 만들 수 있다.
추상 클래스2는 부모 추상 클래스를 구현하지 않아도 된다. extends로 가지고 있다고 여겨지니까..

3. 추상클래스 안에 멤버 변수를 가질 수 있다.

4. 추상 클래스안에 비추상적 메서드를 가질 수 있다.
