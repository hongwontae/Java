상속 생성자
상속받은 클래스가 생성자를 호출할 때 함축적으로 super()라는 메서드가 존재하여 상속 클래스의 생성자를 먼저 호출하고 자신의 생성자를 호출한다. 출력 과정 => 상속클래스 생성자 이후 기존 클래스 생성자
=> 기본 생성자 기준

상위 클래스 생성자(name) 서브 클래스 생성자(title)
상위 클래스에서 생성자를 적고 main 클래스에서 서브 클래스 객체를 생성하면 => 오류가 발생한다.
서브 클래스 객체가 extends하여 다 받았을 것 같지만 생성자는 예외이다. 서브 클래스에서 상위 클래스 생성자를 구현해야만 서브클래스 객체에서 생성자들을 제대로 사용할 수 있다. super(name) => 이렇게
