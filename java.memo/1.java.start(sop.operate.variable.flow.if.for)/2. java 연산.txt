1. 연산 규칙
1-1. 문자열 + 숫자
=> 모두 문자열로 처리됩니다. 즉, 그냥 concat 됩니다.
=> 문자열 + 숫자 + 숫자도 숫자+숫자 처리가 선행되는게 아니라서 다 합쳐집니다. (문자열+숫자가 제일 왼쪽에 있음으로 우선순위)

1-2. 정수/정수 = 정수
=> 정수끼리 나눠서 나머지가 발생하면 나머지는 버리고 정수 값만 반환됩니다. (반환값은 int 타입)
=> 나머지 값을 알려면 타입을 실수 타입으로 변환해야 합니다. (float, double)


2. 연산자 우선순위
=> 괄호> 단항 > 산술 > 비교 > 논리 > 대입


3. 세미콜론 역할
=> 하나의 명령(statement)의 끝을 나타내는 마침표 같은 역할입니다.
=> 변수 선언, 메서드 호출, 할당, 반복 or 조건문 안에서 한 줄 짜리 명령의 끝에는 꼭 붙여야 합니다.
