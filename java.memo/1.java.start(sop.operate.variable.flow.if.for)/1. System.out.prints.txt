1. System.out.print()
=> 줄바꿈 없이 출력하는 메서드입니다.

2. System.out.println()
=> 출력 후 줄바꿈을 해주는 메서드입니다.

3. System.out.printf()
=> 포맷 형식을 지정하여 출력합니다.
=> %d, %s, %f 형식을 지정하여 첫 번쨰 인자에 문자열과 함께 사용하고, 두 번쨰 인자부터 형식의 값을 채웁니다.


4. escape 문자
=> 백슬래시로 시작합니다.
4-1. \n : 줄바꿈
4-2. \t : tap
4-3. \" : 큰 따옴표
4-4. \' : 작은 따옴표
4-5. \\ : 슬래시
4-6. \b : 백스페이스...


5. 만약 printf()을 사용하고 출력문을 정렬하고 싶다면 printf().println()
